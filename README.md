# Hello World TypeScript Project

This is a simple TypeScript project that demonstrates a basic "Hello, World!" application.

## Project Structure

```
hello-world-ts
├── src
│   ├── app.ts
│   └── types
│       └── index.ts
├── package.json
├── tsconfig.json
└── README.md
```

## Getting Started

To get started with this project, follow the steps below:

### Prerequisites

Make sure you have Node.js and npm installed on your machine. You can download them from [nodejs.org](https://nodejs.org/).

### Installation

1. Clone the repository or download the project files.
2. Navigate to the project directory:
   ```
   cd hello-world-ts
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application

To run the application, use the following command:
```
npm start
```

This will compile the TypeScript code and execute the `app.ts` file, which will log "Hello, World!" to the console.
